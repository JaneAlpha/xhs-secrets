## 📖 概述

这是一份完整的小红书运营攻略，涵盖8大核心模块：

- **定位密码** - 找到你的专属赛道
- **内容密码** - 爆款笔记的诞生公式
- **关键词密码** - 让算法找到你
- **流量密码** - 穿透流量池的层级跃迁
- **等级密码** - 账号权重的积累法则
- **变现密码** - 找到你的盈利模式
- **团队密码** - 从单打独斗到规模化运营
- **运营红线** - 必须遵守的禁规则

## 🎯 核心内容

| 模块 | 关键内容 |
|------|---------|
| 选题标准 | 只做互动>5000的选题 |
| 标题公式 | 8种爆款标题模式 |
| 封面公式 | 拼图 + 花字 |
| 流量池 | L1-L4层级跃迁机制 |
| 变现路径 | 0到5000+粉丝的变现方式 |

## 🚀 快速开始

### 在线访问

直接访问：[https://janealpha.github.io/xhs-secrets/](https://janealpha.github.io/xhs-secrets/)

### 本地运行

```bash
# 克隆项目
git clone https://github.com/janealpha/xhs-secrets.git
cd xhs-secrets

# 使用任意静态服务器
# Python
python -m http.server 8080

# Node.js
npx serve

# 或直接打开 dist/index.html
```
